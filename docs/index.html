Windows 11 호환
LAN 메신저, Y-popup

2000년대 추억의 'X-Popup (빨간전화기)' LAN 메신저를 최신 규격으로 오마주했습니다.
같은 네트워크(LAN)에 연결된 PC끼리 복잡한 가입 없이 쪽지와 파일을 빠르게 주고받으세요.

설치 없이 실행하기 (단일 exe) 사용 방법 가이드
🔍

자동 감지 (UDP Broadcast)

계정 가입이나 서버 설정 없이, 실행만 하면 동일 네트워크에 있는 사용자를 자동으로 발견합니다.

✉️

1:1 즉각 팝업 쪽지

수신 즉시 상대방 화면 중앙에 팝업창으로 메시지가 전달되는 직관적인 인스턴트 쪽지 전송 기능입니다.

📁

파일 드래그 앤 드롭 전송

보낼 파일을 쪽지 창 어디에나 드래그하여 놓으면 편리하게 다중 파일 전송이 가능합니다.

🌙

부재중 및 자동 응답

일정 시간 자리 비움 시 자동으로 부재중 상태(IsAway)로 전환되며 커스텀 자동 답장을 전송합니다.

시작하기 안내

01

파일 다운로드

상단의 다운로드 버튼을 눌러 단일 실행 파일 Y-popup.exe를 다운로드합니다.

02

방화벽 규칙 허용

첫 실행 시 방화벽 알림창이 나타나면 반드시 '허용'을 클릭해 주십시오 (포트: UDP 50505, TCP 50506).

03

트레이에서 관리

실행 후 화면 우측 하단 트레이의 빨간 전화기 아이콘을 통해 사용자 목록 및 설정을 열 수 있습니다.
